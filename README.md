## Лабораторна робота №2

**NumPy. Одновимірні масиви**

Програма формує послідовність температур за рік у вигляді одновимірного масиву.
Обчислюється середня температура та визначається кількість днів, у які температура перевищувала середнє значення на **5°C**.

Структура

* `generate_data()` — створення масиву температур
* `process_data()` — обробка даних
* `print_results()` — виведення результатів
* `main()` — запуск програми

Використано базові можливості бібліотеки **NumPy**: генерація випадкових чисел, обчислення середнього значення та булева індексація.
